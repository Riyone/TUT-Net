# TUT-Net
This is the appendix and code for BMVC paper--TuTNet - A method of fabric sewing thread segmentation in textures background

The presence of defects during fabric sewing is a challenging issue in factory quality control. Accurate extraction of the sewing thread area helps detect sewing area defects downstream. In this paper, Twin-UTransformer Net (TUTNet) is proposed, which is a network for sewing thread segmentation combined with U-shaped structures and Transformers. To ensure local edge segmentation accuracy, a local attention module is proposed that uses double convolutions at low layers to maintain computational efficiency while capturing low-dimensional features. To learn high-dimensional abstract features of sewing thread, a global attention module is proposed that calculates global attention through matrix transposition of image patches and captures global context by fusing extracted features from multiple layers. To address data imbalance caused by small target regions, dual loss functions are used to guide model convergence. Finally, two fabric sewing thread datasets are established. Experimental results on multiple real datasets show that our model has good segmentation performance and generalization ability.
